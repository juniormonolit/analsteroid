## 2026-07-03 — Тепловая карта: ранговая (перцентильная) шкала вместо min/max

- Проблема: линейная шкала min→max ломалась об выбросы — менеджер с 3 сделками и 3
  продажами (CR 100%) красил всех остальных (среднебольничные ~12%) в красный.
- Фикс: цвет = РАНГ значения среди всех значений колонки (перцентиль). Медиана — всегда
  середина шкалы (жёлтый), выброс — просто самый зелёный. Равные значения → одинаковый
  цвет (средний ранг связки: (меньших + (равных−1)/2) / (n−1)).
- `heatStats` теперь хранит отсортированный массив значений колонки, не {min,max}.
  Инверсия («меньше = лучше») работает поверх ранга.
- Деплой → BUILD `SI3ynhIJc2KRTSwVX839A`. Миграций нет.
